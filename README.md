# Loan Approval Prediction System

A modern and user-friendly web application for predicting loan approval status using machine learning.

## Features

- Interactive user interface with Streamlit
- Real-time loan approval prediction
- Visual representation of income distribution
- Detailed probability analysis
- Modern and aesthetic design

## Prerequisites

- Python 3.8 or higher
- pip (Python package installer)

## Installation

1. Clone this repository or download the source code
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

1. Make sure you have trained and saved your model as 'loan_model.pkl' in the same directory
2. Run the Streamlit application:
   ```bash
   streamlit run app.py
   ```
3. Open your web browser and navigate to the URL shown in the terminal (usually http://localhost:8501)

## Input Features

- Gender
- Marital Status
- Number of Dependents
- Education Level
- Employment Status
- Applicant Income
- Co-applicant Income
- Loan Amount
- Loan Term
- Credit History
- Property Area

## Output

- Loan Approval Prediction (Approved/Rejected)
- Probability of Approval/Rejection
- Visual representation of income distribution
- Detailed probability analysis
